# DBMS Mini Project

A database management system mini project built with HTML, CSS, and JavaScript.

## Branch Information

This repository uses **main** as the default branch. All pull requests and comparisons should be made against the **main** branch.

**Note:** This repository does not use a master branch. Please ensure all your work is based on and merged into the `main` branch.

## Features

- Event Management System
- User Profile Management
- Event Bookings
- Event Creation

## Project Structure

```
dbsm mini/
├── index.html          # Landing page
├── home.html           # Home page
├── events.html         # Events listing
├── bookings.html       # Bookings management
├── profile.html        # User profile
├── create-event.html   # Event creation
├── css/                # Stylesheets
│   ├── navbar.css
│   └── home.css
└── js/                 # JavaScript files
    ├── auth.js
    └── main.js
```

## Getting Started

Simply open the `index.html` file in a web browser to start using the application.

## Contributing

When contributing to this repository:

1. Fork the repository
2. Create your feature branch from `main`: `git checkout -b feature/my-feature main`
3. Commit your changes: `git commit -m 'Add some feature'`
4. Push to the branch: `git push origin feature/my-feature`
5. Open a Pull Request against the `main` branch

## CI/CD

This repository includes GitHub Actions workflows that run on pushes and pull requests to the `main` branch.
